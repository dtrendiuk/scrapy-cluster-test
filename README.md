# scrapy-cluster
## Development:
1. Install dev packages:
```bash
pip install -r ./requirements-dev.txt
```
2. Install pre-commit hook
```bash
pre-commit install
```
## Testing
- Run:
```bash
pip install -r ./requirements-dev.txt
pytest
```
